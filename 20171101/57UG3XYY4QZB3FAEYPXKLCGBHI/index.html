八達通推QR Code　擦鞋檔：少人用

【本報訊】八達通推二維碼（QR Code）支付，原難負擔電子支付的小攤檔，上周亦已試行，近日中環戲院里數間擦鞋檔放上了八達通機及二維碼紙板，擦鞋師英姐受訪時指，多數人仍使用八達通卡，二維碼使用量較少。
本報記者在擦鞋檔實測二維碼支付，英姐幫記者的一雙皮鞋擦得亮晶晶後，記者打開八達通應用程式（App）內的好易畀（O!ePay）功能，按「付款」後，右上角有二維碼掃描器，掃描英姐獨有的二維碼後，即有填寫金額指示，輸入50元後，資金即過賬。不過，英姐若想了解資金是否已過賬，須先開啟其手機，並打開八達通商戶App，才會顯示資金是否已過賬，否則較難稽考，因沒有八達通卡支付般「嘟」一聲的提示功能。

用八達通卡更方便

由於八達通尚未正式公佈服務已開通，故此懂使用的人不多，而英姐亦應記者要求時，才開啟手機及接納二維碼支付，八達通卡反而更方便快捷。
事實上，八達通為增加商戶採用誘因，二維碼紙板外附送八達通讀卡器（即POS機），一般客戶可以使用八達通卡付款。八達通發言人指出，現階段豁免參予商戶首年費用。不過據悉首年免費後，第二年開始要收1.5%。
英姐讚八達通卡機好用，「一日十幾廿單用八達通嘟，我哋小商戶都要趕電子支付呢個潮流」。記者現場所見，不少人採用八達通卡付款多於現金，故此即使往後要收手續費，英姐亦會考慮採用。至於二維碼支付，她透露是1個月結算1次，而且1,000元以上才會結算。
此外，透過二維碼付款後，八達通App將會顯示支付紀錄，包括交易時間、賬戶號碼、參考編號，而收據將會保存270天。
八達通長遠亦要解決O!ePay增值問題，以iPhone為例，現時可透過銀行轉賬及7-11增值，惟未開放近場通訊（NFC）功能，未能透過八達通卡幫O!ePay增值，除非買入八達通的藍芽連接器，Android機則可免卻此問題。
