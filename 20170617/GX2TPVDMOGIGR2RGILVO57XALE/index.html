【鳩嗚】為慈善你敢脫麼？
首次Bra Top赤膊路跑賽

跑步時男士赤膊上身、女士只著Sports Bra，係咪好養眼、好無拘無束呢？費清講緊嘅係首次喺香港搞嘅Chubb YOLO Run HK 2017，之前喺新加坡已經舉辦咗兩屆，鼓勵參加者脫去束縛，釋放自我，無論係排骨身型，定擁有一個大肚腩，都唔洗介懷，希望參加者可以「大解脫」，唔好理其他人想法，齊齊跑出正能量，仲可以做善事。

YOLO Run嘅YOLO，意思係you only live once，即係活在當下。跑步活動會係10月1日香港科技園舉行，比賽時間由上午8時至中午12時，分為5公里及10公里賽事，只有10公里屬於競賽，而所有參加者完成賽事後仲可以參加大型瑜珈班放鬆一下。每位赤裸上身嘅男參賽者，以及只著Sports Bra嘅女參賽者，主辦單位都會捐出$20比國際培幼會（香港）作慈善用途。

活動由「動起來」舉辦，主辦單位認為，除衫跑步對某啲人嚟講，已經好YOLO下。新加坡由2015年首辦以來，已經累積過萬名參加者，雖然唔係硬性要跑手除衫，但新加坡嘅參賽者都有兩成人真係會赤膊去參賽。

7月10日前報告會有早鳥優惠，5公里報名費300元，10公里就要350元。每名參加者會有一份選手包，包括T恤、背心、運動內衣及繩索袋，所有參賽者完成賽事後，亦可獲得YOLO Run獎牌紀念。詳情可瀏覽 http://www.yolorun.com/hongkong 。
